\# 油松雄球花物候期识别系统



基于 YOLOv11 的油松雄球花物候期自动识别系统，可识别伸长期、成熟期、衰退期三个阶段。



\## 功能特点

\- 🖼️ 支持图片识别

\- 🎥 支持视频识别  

\- 📊 实时统计检测结果

\- 🎯 高精度 YOLOv11 模型



\## 使用方法

1\. 上传图片或视频文件

2\. 点击"开始检测"按钮

3\. 查看检测结果和统计信息



\## 技术架构

\- YOLOv11 目标检测

\- Streamlit Web框架

\- PyTorch 深度学习

\- OpenCV 图像处理



\## 物候期类别

\- 伸长期 (Elongation Stage)

\- 成熟期 (Ripening Stage) 

\- 衰退期 (Decline Stage)
